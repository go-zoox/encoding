## Installation
To install the package, run:
```bash
go get github.com/go-zoox/fetch
```

## Getting Started

```go
response, _ := fetch.Get("https://httpbin.zcorky.com/get")
url := response.Get("url")
method := response.Get("method")

fmt.Println(url, method)
```
